# 技術堆疊

## 後端

技術堆疊將根據實作決定。

常見模式：
- RESTful 或 GraphQL API 設計
- 資料庫整合
- 身份驗證和授權
- 錯誤處理和日誌記錄

## 前端

技術堆疊將根據實作決定。

常見模式：
- 基於元件的架構
- 狀態管理
- API 整合
- 響應式設計

## 常用指令

指令將隨著專案開發而新增。典型指令包括：

### 後端
```
# 安裝依賴
npm install / pip install / 等

# 執行開發伺服器
npm run dev / python manage.py runserver / 等

# 執行測試
npm test / pytest / 等
```

### 前端
```
# 安裝依賴
npm install / yarn install

# 執行開發伺服器
npm run dev / yarn dev

# 建置生產版本
npm run build / yarn build

# 執行測試
npm test / yarn test
```
